本书特色

[

《父亲快跑》是一部同名散文、中篇小说和电影文学剧本的合集，所写人物、事件也都是一致的，主人公的原型是作者父亲、抗日小英雄园子。书稿采用白描的手法、诗性的语言，记叙了园子与战友二牛、春生、秀姑等小八路群体，不折不挠、前仆后继抗击日本侵略者的光辉事迹，可歌可泣，可圈可点。这个英雄群体先后转战河北霸县、白洋淀湖区、冀中大平原等地，艰苦卓绝地抗击日本侵略者。人物栩栩如生，情节步步惊心，叙事娓娓动人。

内容简介

[

当代作家赵万里先生穿越七十年时空，深情回望父亲园子（主人公原型）的少年时代，探寻其在枪林弹雨中飞奔、在血雨腥风里拼杀的身影。 这是一部同名散文、同名小说和同名电影剧本的合集，三种题材一咏三唱，激情澎湃；一瀑三叠，壮怀激烈。作品用条分缕析的笔致、情真意切的文字，把一个个小英雄的成长史、血泪史和抗战史写得骨架结实，血肉丰满，令人唏嘘之余，情难自已地掩卷深省。 作家赵万里以一条大河的奔流，象征民族的生生不息；以一个家庭的劫难，抒写灵魂的苦苦挣扎，堪为民族心史上一通厚重的碑石。 书中既有铁与火，也有血与泪，更交织着国恨家仇。这样饱蕴家国情怀的故事，不仅有利于丰满我们的民族魂，而且有利于锻造我们的中国梦；不仅是我们每一个中国人必须补上的爱国主义教育课，更是反对日本军国主义死灰复燃必须吹响的冲锋号。

]

作者简介

[

　　赵万里，当代作家，诗人。1958年生，祖籍河北定兴，现居北京。主要著作有：散文集《静水流深》，诗集《生命辉煌》，长篇小说《天安门》、《西柏坡》等，即将出版长篇小说《短兵》等。文学作品多篇入选中学、大学教材和教辅读本，部分作品被译介到海外；散文《父亲快跑》被《读者》杂志选发在2010年第19期的头条。

]
