内容简介

[

本书研究欠驱动航天器编队重构问题, 分析了径向或迹向欠驱动条件下圆轨道以及椭圆轨道编队重构的可行性。基于可行性分析结果, 创新提出一类降维滑模控制理论与方法, 间接实现了径向或迹向欠驱动的航天器编队重构控制。同时, 本书首次解析推导了圆轨道欠驱动编队重构*优解析解, 并首次设计了欠驱动避撞机动策略。全书共分为六章, **章综述欠驱动编队重构研究进展 ; 第二章介绍圆轨道欠驱动编队重构渐近稳定控制方法, 第三章首次解析推导了圆轨道欠驱动编队重构*优解析解, 并设计了闭环自适应控制器 ; 第四章首次设计了新型欠驱动避撞机动策略以及欠驱动避撞重构控制器 ; 第五章介绍椭圆轨道欠驱动编队重构自适应渐近稳定控制方法。
