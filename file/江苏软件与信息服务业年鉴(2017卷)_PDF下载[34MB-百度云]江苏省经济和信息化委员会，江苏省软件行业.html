内容简介

本书内容共分为六部分: **部分为产业综述篇, 介绍2016年江苏省及全国软件产业发展概况 ; 第二部分为地市产业篇, 介绍全省13个地市2016年软件产业发展情况 ; 第三部分为园区产业篇, 介绍省内重点软件园区2016年发展情况 ; 第四部分为产业政策篇, 介绍2016年发布的国家级及省级软件产业政策 ; 第五部分为产业要素篇, 介绍2016年省内软件企业、人才、软件产品、相关技术发展情况 ; 第六部分为产业大事记, 介绍省内2016年为发展软件和信息技术服务业开展的重要活动, 以及有影响力的工作。
