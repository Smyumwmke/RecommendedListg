本书特色

[

《游离书》是第六届（2014）鲁迅文学奖得主海男思想闪光的结集，如同海男当年的《男人传》，依然很难将《游离书》纳入某种约定束成的文本之中。
　　是她在*闲暇的某一个瞬间、*专注于某一事物的瞬间、*感伤和*快乐的某一个瞬间……在田园，在山冈，在她仰起头来仰望天空中一朵飘浮云瞬间……世界破碎如斯，上帝派她来悟道语词，她必须躬下身去，重拾碎片，将它们串成珍珠，去照亮并温暖人心。当一个人专注某一个瞬间，那些生命中微小的事件和她的思想碰撞，火花犹如夜空中的流星一样——它只呈现给那些夜晚思索的人、迷途的人。
　　语言是出卖思想的，这是一个作家*为真实的部分，这更接近于人性。无数的瞬间，让海男充当了那些迷途者的代言，她不由自主地发声：“好好陪你身边的*爱吧!幸福是属于你的。请别相信我语言中的谎言，世界在加速沙漠化的同时，也在加速谎言的力度。”她又说：“回忆，仿佛被行将焚尽的烟蒂烙伤的手，现在，那些苍茫时空的回忆将我带到了哪里？而我的手，在此伸出去，是否可以就此抚摸到那些致命的忧伤，它们在我身体的疆域中千回百转，只有在它们遇上剑一样逼近的山峦江流，它们才会通灵于那些忍不住的吟唱……”如此，在破碎之中，悟道的语词的海男只有大声地喊出箴言，她也喊出了作为个体生命活着的真理：“我能成为自己，那是因为我的头顶有一股缥缈的青烟。有你，固化的也能变成一缕轻雾，我就是我，接纳不代表苟同。”她和语凡对话，谈论哈尼梯田的山水，她说战争和项羽，谈论刺客和鲜血……她就是那个精神的呓语者，在黑夜中站出来，她所呼喊出来的，来自于她的思索和*真实的内心。
　　《游离书》就是这样一部箴言之书，那些弥漫于“海氏语词”中的箴言，像白银一样穿透人心。

]

内容简介

[

1．所谓游离，就是有所恃，无所求；所谓诗人，就是“游离份子”。在《游离书》中，海男以诗性的语言，诠释时空和轮回。虽然我们只活在瞬间，但灵魂想要抵达的地方，终会抵达。
　　2．鲁迅文学奖获奖作家作品。富有争议的女性主义作家，第六届鲁迅文学奖获奖作家海男代表作品，讲述游离于红尘内外的生命箴言。
　　3.时间的永恒和沉默，肉身的短暂和美丽，是海男的写作主题。《游离书》中充斥着发自灵魂深处的呓语，像一串晶莹剔透的水晶珠链，让人爱不释手。

作者简介

[

海男，女；出生于二十世纪六十年代，中国当代著名作家，中国女性先锋作家代表人之一。曾获1996年刘丽安诗歌奖；中国新时期十大女诗人殊荣奖；2005年《诗歌报》年度诗人奖；2008年《诗歌月刊》实力派诗人奖；2009年荣获第三届中国女性文学奖；2014年获第六届鲁迅文学奖。
海男的跨文本写作《男人传》《女人传》《身体传》《爱情传》等；长篇小说代表作《花纹》《夜生活》《马帮城》《私生活》；散文集《空中花园》《屏风中的声音》《我的魔法之旅》《请男人干杯》等；诗歌集《唇色》《虚构的玫瑰》《是什么在背后》等。现为云南师范大学特聘教授。
