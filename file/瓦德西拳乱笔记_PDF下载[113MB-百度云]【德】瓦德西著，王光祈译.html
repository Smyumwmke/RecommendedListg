内容简介

[

　　《瓦德西拳乱笔记》按时间顺序，收有自1900年8月至1901年11月期间，八国联军侵华战争的联军统帅瓦德西所写的日记、笔记，给德皇的奏议、函电、报告，以及德皇的谕旨、函电等，记述和反映了八国联军在华侵略活动及其内部矛盾、镇压义和团运动、胁迫清政府接受议和大纲，以及八国军队烧杀抢掠等内容，对了解和研究八国联军侵华战争与义和团运动有一定的参考价值，具有一定史料研究价值。 　　本书作者瓦德西，是1900年八国联军侵华战争时的联军统帅，所以此次出版的《瓦德西拳乱笔记》具有日记性质，真实性强，可以更好地再现那段难忘的历史，以史为鉴，对现代具有重要的意义。

]

作者简介

[

　　阿尔弗雷德·格拉夫·冯·瓦德西（Alfred Graf Von Waldersee），伯爵，德国元帅。早年入普鲁士边防炮队，普法战争期间，担任国王的副官。后曾陆续担任陆军总参谋长、第九军军长。1900年，中国发生义和团运动，瓦德西出任八国联军总司令，被德皇授予元帅军衔。1901年6月离开北京，1904年病逝。著有《瓦德西回忆录》。 　　王光祈，音乐学家和社会活动家。他的《欧洲音乐进化论》、《德国国民学校与唱歌》、《东西乐制之研究》，是中国最早的、具有开创意义的比较音乐学著作。他的研究开东方民族音乐之先河，他曾为1929年版的《不列颠百科全书》和《意大利百科全书》撰写《中国音乐》专稿。他还编译了政论著作《辛亥革命与列强态度》、《三国干涉还辽秘闻》等。

]
