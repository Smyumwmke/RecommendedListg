本书特色

[

德国是作者度过了十多年的留学生活，生活的国家，深切地体会到了文化和价值观之间的矛盾。他用漫画的形式多信息量解读了德国。他用漫画的笔触呈现德国人的性格，用今天的视角折射欧洲的历史。刻板严谨的德国人，为何格外讲究原则和秩序？同为欧洲人，德国人为何与自由浪漫的法国人差异巨大？铁血宰相俾斯麦是如何把的郭建成为称霸欧洲的德意志帝国的？酷爱香肠和啤酒的德国人为什么能生产出奔驰和宝马？为什么大文豪、哲学家、音乐家会层出不穷？歌德、席勒、贝多芬、莫扎特、康德、尼采、黑格尔群星闪烁。莱茵河见证了日耳曼民族的诞生，成长和曲折发展的历程，铸就了德意志工业文明的奇迹。

作者简介

[

李元馥，1946年出生于韩国忠南大田。自首尔大学工科学院建筑学系毕业后，李元馥走上了漫画家和设计师的独特职业生涯之路。1975年，他赴德国明斯特大学留学，主修设计专业，获设计专业学位，并获"校长奖"。他曾在德国举办个人展览。现任德成女子大学产业美术系教授、韩国漫画和动画学会会长，也是美国加州大学尔湾分校的客座教授。著有《教育漫话韩国史》、《漫话世界》系列等多部漫画著作，其作品主要通过既通俗又有趣的语言介绍世界的历史、文化以及经济等。他对韩国漫画文化的贡献受到人们的高度认可，并在1993年获得第九届韩国大田国际奖。
